mBase-Cli
============
> CLI Tool for mBase

#Usage

1. Run `npm install -g mb-cli` to install mBase-Cli

2. Config `package.json` in your local project   

example:

```
  "mBase": [
    "components/banner.less",
    "elements/arrow.less",
    "elements/button.less"
  ]

```

2. Run `mb-cli` and mBase-Cli will do all the things left
